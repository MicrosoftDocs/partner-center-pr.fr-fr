# <a name="manage-your-partner-center-account---company-profile-bank-information-roles-permissions-and-more"></a>Gérer votre compte Espace partenaires : profil de société, informations bancaires, rôles, autorisations, etc.

**Rôles appropriés**

- administrateur général
- Administrateur MPN

Le compte que vous créez dans l’Espace partenaires au moment de l’inscription comprend votre adresse e-mail professionnelle et des informations sur votre entreprise. Une fois que vous avez créé un compte dans l’Espace partenaires, vous devez achever la configuration du profil complet de votre société en incluant les détails du support, les exonérations fiscales le cas échéant ainsi que les informations de l’interlocuteur principal. Le compte de votre entreprise inclut également les comptes d’utilisateur de toute personne devant utiliser l’Espace partenaires pour ajouter ou gérer des clients, vendre des abonnements, travailler à la facturation, créer des profils métier, gérer les références, travailler sur des programmes d’incentives et assurer un support.

Voir aussi 

- [Inviter des employés à rejoindre l’Espace partenaires](guide-to-migration.md)

- [Ajouter un nouvel utilisateur](create-user-accounts-and-set-permissions.md)

>[!NOTE]
>Le menu des paramètres du compte change en fonction des programmes auxquels vous êtes inscrit. Par exemple, si vous êtes partenaire du programme Fournisseur de solutions Cloud, vous pouvez gérer votre offre Azure Partner Shared Services à partir des paramètres du compte.

## <a name="account-management"></a>Account management

Pour accéder à la gestion des comptes de tous les partenaires (commerciaux ou développeurs) dans l’Espace partenaires, connectez-vous au [tableau de bord](https://partner.microsoft.com/dashboard) de l’Espace partenaires, puis sélectionnez l’option **Paramètres** (icône d’engrenage) dans le coin supérieur droit de l’écran. Vous n’avez plus à choisir entre Partenaire et Développeur, car tous les profils et paramètres sont désormais combinés.

:::image type="content" source="images/accountsettings/opening.png" alt-text="Icône des paramètres":::


:::image type="content" source="images/accountsettings/accountsettingssv.png" alt-text="Paramètres de compte":::

### <a name="locate-your-mpn-id"></a>Localiser votre ID MPN

Que vous soyez partenaire ou développeur, vous devez utiliser votre ID MPN pour par exemple consigner des tickets de support. Votre ID MPN se trouve sur la page des **informations légales**. Vous pouvez également y accéder via le **profil d’identité**.

## <a name="organization-profile"></a>Profil de l’organisation

Vous gérez votre profil métier légal, votre profil de partenaire et les localisations de votre société sur la page **Profil de l’organisation**. Cette page vous permet de voir les informations de premier niveau sur votre entreprise. Vous pouvez explorer et modifier chaque zone. Sélectionnez l’onglet correspondant au programme qui vous intéresse (MPN, Fournisseur de solutions Cloud ou Place de marché commerciale). Ainsi, vous verrez les informations qui vous concernent ou qui sont liées au statut du processus de vérification.
 
:::image type="content" source="images/accountsettings/accountsettingsnew.png" alt-text="informations légales":::
 
### <a name="legal-information"></a>Informations légales

Quand vous rejoignez l’Espace partenaires, votre société est soumise à un processus de vérification par Microsoft. Vous pouvez effectuer le suivi du statut de cette vérification dans la page **Profil légal**. L’interlocuteur principal, c’est-à-dire la personne que Microsoft va contacter pour toute question relative au partenaire, et l’interlocuteur légal principal, c’est-à-dire la personne qui gère les aspects juridiques, se trouvent sur la page **Profil légal**. Toutes les localisations commerciales de votre société se trouvent ici. Vous pouvez y ajouter des localisations supplémentaires.  

Découvrez plus en détail la [vérification et les informations associées à votre compte](verification-responses.md).

### <a name="tenants-profile"></a>Profil des locataires

Toutes les informations relatives à vos locataires Azure AD, qu’il s’agisse d’informations commerciales ou liées aux développeurs, se trouvent sur la page **Profil du locataire**. Ce profil est l’emplacement où l’administrateur général peut associer de nouveaux locataires au compte global du partenaire.

### <a name="identifiers"></a>Identificateurs 

La section **Identificateurs** contient les identités de l’Espace partenaires relatives à votre société : ID MPN, ID d’éditeur, ID d’éditeur Windows, etc. Vous pouvez développer et modifier chaque zone, par exemple pour faciliter la localisation de l’interlocuteur principal de votre activité de publication.

### <a name="company-profile"></a>Profil de l’entreprise

Le **Profil de la société** identifie le type de partenariat que vous avez avec Microsoft, par exemple éditeur de logiciels indépendant ou partenaire du programme Fournisseur de solutions Cloud. Vous pouvez voir une capture instantanée du nombre de clients avec lesquels vous travaillez, de votre chiffre d’affaires annuel et de la taille actuelle de votre société. Vous pouvez également développer les informations relatives à la société dans la page des détails de la société. Dans cette page, vous pouvez également indiquer à Microsoft le type de travail que vous souhaitez effectuer, par exemple créer des applications, revendre des logiciels Microsoft et tiers, ou devenir intégrateur de systèmes. Vous pouvez également définir les lieux où vous exercez votre activité ainsi que les régions dans lesquelles vous souhaitez la développer.

### <a name="account-merge-profile"></a>Profil de fusion de compte

Quand vous invitez une société ayant un compte actif dans l’Espace partenaires à fusionner son compte avec le vôtre, vous pouvez gérer les informations correspondantes dans la page **Fusion de compte**. Recherchez l’ID MPN de la société que vous souhaitez inviter à fusionner, visualisez les fusions actuelles et envoyez des invitations aux sociétés appropriées. C’est également là que vous pouvez accepter ou refuser une invitation à fusionner votre compte de société avec celui d’une autre société. Pour obtenir des informations détaillées sur la fusion de comptes, consultez l’article [Fusionner votre compte partenaire avec un autre compte partenaire](merge-accounts.md).

### <a name="payout-and-tax"></a>Revenu et taxe 

La page **Revenu et taxe** contient les détails relatifs à votre chiffre d’affaires et vos taxes ainsi que les informations **Facturer à** et **Numéro de BDC**. Les informations d’identification fiscale de votre société se trouvent également sur cette page, notamment votre numéro de TVA, si vous en avez un, et la devise utilisée.


## <a name="user-roles-and-permissions"></a>Rôles et autorisations d’utilisateur

Votre mode de travail dans l’Espace partenaires et les zones que vous pouvez voir ou mettre à jour dépendent de votre rôle et des autorisations attachées à ce rôle. Par exemple, si vous n’êtes pas administrateur d’incentives, vous ne pouvez rien changer dans les pages Primes incitatives, mais vous pouvez éventuellement voir les données. 

### <a name="update-preferred-email"></a>Mettre à jour l’e-mail par défaut

Pour mettre à jour votre adresse e-mail par défaut afin de recevoir les notifications de l’Espace partenaires : 

1. Connectez-vous à votre [tableau de bord de l’Espace partenaires](https://partner.microsoft.com/dashboard).
1. Accédez aux [**préférences relatives à l’engagement**](https://partner.microsoft.com/dashboard/engagement/preference), sélectionnez le lien **Changer**, mettez à jour l’adresse e-mail, puis cliquez sur **Enregistrer**.

### <a name="find-your-user-role"></a>Trouver votre rôle utilisateur

1. Accédez à **Paramètres**, puis sélectionnez **Mon profil**.
 
1. Dans la page **Détails**, vous voyez vos informations personnelles : travail, formation, examens et certifications.
 
1. Sélectionnez l’onglet **Rôles et autorisations**. Ici, vous voyez tous les rôles qui vous ont été attribués et toutes les autorisations octroyées par ces rôles.

## <a name="programs-in-which-you-are-enrolled"></a>Programmes dans lesquels vous êtes inscrit

La majeure partie du travail que vous faites pour gérer votre compte Espace partenaires est liée aux programmes spécifiques auxquels vous êtes inscrit ainsi qu’aux rôles utilisateur et aux autorisations qui vous ont été attribués dans l’Espace partenaires.

Pour voir à quels programmes vous avez accès dans l’Espace partenaires :

1. Connectez-vous au [tableau de bord](https://partner.microsoft.com/dashboard) de l’Espace partenaires.

2. Le menu de navigation de gauche reflète les programmes auxquels vous êtes inscrit, et affiche les options disponibles pour chaque programme.

Voici un exemple. Supposons qu’un utilisateur a accès à trois programmes dans le tableau de bord de l’Espace partenaires : Le programme **MPN**, le programme **Références** et le programme **Place de marché commerciale**. Quand il se connecte au tableau de bord de l’Espace partenaires, il voit ces trois programmes dans la partie gauche de l’écran.

:::image type="content" source="images/accountsettings/programs-enrolled-left-nav.png" alt-text="menu des programmes auxquels l’utilisateur est inscrit":::

### <a name="enrolling-in-programs"></a>Inscription à des programmes

Les programmes disponibles dans l’Espace partenaires sont nombreux. Chaque programme a ses propres exigences que votre entreprise doit respecter pour pouvoir s’y inscrire.

Pour en savoir plus sur l’inscription à certains programmes de l’Espace partenaires, consultez la liste (non exhaustive) suivante :

- [Programme Place de marché commerciale](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account)
- [Avantages de l’adhésion à Microsoft Partner Network](mpn-overview.md)
- [Programme Fournisseur de solutions Cloud (CSP)](https://docs.microsoft.com/partner-center/enrolling-in-the-csp-program)
- [Office Store](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/office)

Pour en savoir plus sur l’inscription aux programmes de l’Espace partenaires, consultez aussi [Ressources du réseau Partenaires](https://partner.microsoft.com/).
