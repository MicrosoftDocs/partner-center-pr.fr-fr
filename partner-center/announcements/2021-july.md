## <a name="windows-7-extended-security-updates-esu-year-2-2021-product-key-issue-resolved"></a><a name="3"></a>Mises à jour de sécurité étendues de Windows 7 Année 2 (2021) - Problème de clé de produit (Product Key) résolu

### <a name="categories"></a>Catégories

- Date : 02/07/2021
- Fonctions
 
### <a name="impacted-audience"></a>Public concerné

Tous les partenaires qui ont acheté les mises à jour de sécurité étendues de Windows 7 Année 2 (2021) entre le 18 mai 2021 et le 10 juin 2021

### <a name="details"></a>Détails

Un problème temporaire a été identifié pour les mises à jour de sécurité étendues de Windows 7 Année 2 (2021). Pour les commandes passées entre le 18 mai 2021 et le 10 juin 2021, les partenaires/clients ont peut-être reçu une clé de produit pour la référence SKU de l’année 1 (2020). Ce problème a été résolu. Toutes les commandes passées après le 10 juin 2021 ont reçu la bonne clé pour les mises à jour de sécurité étendues de Windows 7 Année 2 (2021). Pour les commandes passées pendant la période concernée, les partenaires doivent retourner l’ancienne commande et faire un nouvel achat.

### <a name="next-steps"></a>Étapes suivantes

Pour commencer le processus de nouvel achat des mises à jour de sécurité étendues de Windows 7 Année 2 (2021), consultez [Suspendre ou annuler un abonnement](../create-a-new-subscription.md#suspend-or-cancel-a-subscription).

### <a name="questions"></a>Vous avez des questions ?

Pour toute question relative à cette notification, contactez l’[Assistance de l’Espace partenaires](https://partner.microsoft.com/support/?stage=1).

## <a name="dynamics-365-cloud-migration-promotion-and-retirement-of-discounted-offers"></a><a name="1"></a>Promotion sur Dynamics 365 Cloud Migration et retrait des offres remisées

### <a name="categories"></a>Catégories

- Date : 01/07/2021
- Fonctions

### <a name="summary"></a>Résumé

Microsoft a le plaisir de vous annoncer le lancement de la promotion sur Dynamics 365 Cloud Migration le 1er août 2021.

### <a name="impacted-audience"></a>Public concerné

Partenaires fournisseurs de solutions Cloud, partenaires locaux Dynamics et fournisseurs de service en couche (LSP)

### <a name="details"></a>Détails

Par le biais du programme des fournisseurs de solutions cloud, les partenaires peuvent transformer numériquement les entreprises de leurs clients avec des solutions cloud leaders sur le marché. Dans le cadre de cet effort, nous mettons actuellement hors service les références SKU existantes (consultez l’onglet des SKU mises hors service dans le [fichier Excel correspondant](https://partner.microsoft.com/resources/detail/dynamics-365-cloud-promotion-retired-skus-xls)) le 31 juillet 2021 et de les remplacer par les nouvelles SKU de la promotion Dynamics 365 Cloud Migration.

Microsoft a le plaisir de vous annoncer le lancement de la promotion sur Dynamics 365 Cloud Migration le 1er août 2021. Cette nouvelle promotion propose des services en ligne Dynamics 365 à prix réduits pour accompagner la migration des clients vers le cloud. Elle concerne également les clients qui ont commencé leur transformation numérique dans le cadre des offres de migration héritées pour passer aux nouveaux prix promotionnels.

#### <a name="dynamics-365-cloud-migration-promotion"></a>Promotion de Dynamics 365 Cloud Migration

**Avantages**

La promotion Cloud Migration offre les avantages suivants aux clients :  

- Remise de 40 % sur les services Dynamics 365 Online avec un abonnement de trois ans, assurant la prévisibilité des prix lors de la migration vers le cloud.

- Droits d’accès double (utilisation dans le cloud et locale simultanée), prenant en charge l’accès utilisateur avant la fin de la solution locale une fois la migration cloud terminée.

- Prise en charge des migrations cloud échelonnées, permettant aux organisations d’ajouter de nouveaux utilisateurs pendant toute la période de promotion.

**Conditions générales**

Programme de licence : CSP

Produit : Promotion de Dynamics 365 Cloud Migration

Remise promotionnelle : 40 %

Durée de la promotion : du 1er août 2021 au 30 juin 2023

Condition d’abonnement : abonnements de trois ans ; le prix promotionnel n’est pas renouvelable.

Licences requises :

- Toutes les licences locales Dynamics (Dynamics AX, CRM, partenaire Governement, NAV, licence d’abonnement, etc.) avec un plan de support et de services clients Dynamics actifs (par exemple, Enhancement Plan ou Advantage Plan) ou Software Assurance (SA) et se renouvelant dans le cadre des services en ligne Dynamics 365.

- Tous les services en ligne Dynamics 365 avec des abonnements actifs pour Dynamics 365 à partir de SA ou de la liste de prix Dynamics, du complément Dynamics 365 Cloud, de l’offre Dynamics 365 Qualified ou de l’offre Dynamics 365 Cloud Migration.

- Cette promotion n’est pas applicable aux nouveaux clients cloud ni aux clients locaux Dynamics inactifs. (Reportez-vous aux [stratégies des services en ligne](https://www.microsoft.com/licensing/terms/productoffering/MicrosoftDynamics365Services/EAEAS).)

**Références SKU**

Pour obtenir la liste complète des SKU, consultez l’onglet des SKU promotionnelles dans le [fichier Excel correspondant](https://partner.microsoft.com/resources/detail/dynamics-365-cloud-promotion-retired-skus-xls)).
