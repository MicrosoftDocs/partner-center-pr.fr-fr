## <a name="new-microsoft-365-enterprise-voice-skus-for-teams"></a><a name="4"></a>Nouvelles références SKU Téléphonie Microsoft 365 Entreprise pour Teams

### <a name="categories"></a>Catégories

- Date : 02-07-2020
- Tapez : Offres/marchés
- État : Bientôt disponible

### <a name="summary"></a>Résumé

Le 1er août 2020, Microsoft introduira une nouvelle offre Téléphonie Microsoft 365 Entreprise pour Teams incluant les composants Système téléphonique, Audioconférence et Forfaits d’appels nationaux.

### <a name="impacted-audience"></a>Public concerné

Direct Provider, Indirect Provider, Indirect Reseller 

### <a name="details"></a>Détails

Le 1er août 2020, Microsoft introduira de nouvelles références SKU Téléphonie Microsoft 365 Entreprise. Téléphonie Microsoft 365 Entreprise est une solution de téléphonie basée sur le cloud qui permet aux utilisateurs de passer et de recevoir des appels RTC dans Microsoft Teams. Cette solution inclut les composants Système téléphonique, Audioconférence et Forfaits d’appels nationaux.

E5 recevra 120 minutes d’appels nationaux sortants, soit l’équivalent de Téléphonie Entreprise - Plan 1.

Les nouvelles offres seront disponibles dans tous les [pays du forfait d’appels](https://docs.microsoft.com/MicrosoftTeams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans), à l’exception des États-Unis et de Porto Rico pour le moment. Dès que nous aurons des informations sur la disponibilité aux États-Unis, nous vous les communiquerons.

**De quoi s’agit-il ?**

Téléphonie Entreprise est une nouvelle offre qui inclut les composants Système téléphonique et Audioconférence ainsi que des minutes pour passer des appels nationaux. 

**Combien ça coûte ?**

Les prix des nouvelles offres figurent dans la liste de prix préliminaire d’août qui se trouve dans la page **Tarifs et offres** de l’Espace partenaires.
        
**Quelle est la date de lancement prévue ?**

Les offres seront normalement disponibles le 1er août 2020. La liste de prix préliminaire d’août contient des informations supplémentaires sur les offres.  

**Quels sont les clients ciblés ?**

- Organisations commerciales, du secteur public commercial international et à but non lucratif
- Les clients du secteur de l’éducation n’y auront pas accès. La disponibilité pour les clients GCC, GCC-High et DoD n’a pas encore été déterminée.

**Comment acheter ?**

Accord Entreprise (EA)/Abonnement Entreprise (EAS), Fournisseur de solutions Cloud ou Direct sur le web

**Où vendre ?**

Dans tous les pays vendant actuellement des forfaits d’appels, **à l’exception des États-Unis et de Porto Rico**.

**Quels sont les prérequis ?**

M365 F3/E3 ou O365 F3/E1/E3

**Les plans existants offrent-ils une valeur ajoutée ?**

Les plans M365/O365 E5/A5 recevront 120 minutes d’appels nationaux sortants, soit l’équivalent du Plan 1. Ces minutes seront ajoutées sans frais supplémentaires, mais nécessiteront une nouvelle référence SKU Microsoft 365/Office 365 E5/A5 « avec minutes d’appel » pour activer les minutes d’appel.

## <a name="microsoft-has-ended-sales-of-the-microsoft-kaizala-pro-standalone-sku"></a><a name="2"></a>Microsoft a mis fin aux ventes de la référence SKU autonome Microsoft Kaizala Pro

### <a name="categories"></a>Catégories

- Date : 01-07-2020
- Tapez : Offres/marchés
- État : Disponible immédiatement

### <a name="summary"></a>Résumé

Microsoft Kaizala Pro n’est plus disponible pour les nouveaux clients ou dans le cadre de renouvellements. Pour accéder à Kaizala, ces clients peuvent passer à d’autres licences Microsoft 365 et Office 365.

### <a name="impacted-audience"></a>Public concerné

Partenaires vendant la référence autonome Microsoft Kaizala Pro

### <a name="details"></a>Détails

Microsoft s’emploie actuellement à ajouter quelques-unes des fonctionnalités les plus utilisées de Kaizala Pro à Microsoft Teams, le hub de Microsoft 365 pour le travail en équipe.

À compter d’aujourd’hui, 1er juillet 2020, Microsoft met fin aux ventes de la référence SKU autonome Kaizala Pro pour les nouveaux clients ou dans le cadre de renouvellements. Les clients existants peuvent continuer à accéder au service par le biais de leur licence actuelle. Nous avons annoncé ce changement aux clients de Kaizala le 1er juin 2020.

Kaizala et Teams continueront d’être accessibles dans le cadre des références SKU Microsoft 365 et Office 365 suivantes :

[Microsoft 365 pour les entreprises](https://www.microsoft.com/microsoft-365/compare-all-microsoft-365-products?&activetab=tab%3aprimaryr2), notamment :  
- Microsoft 365 Business Basic
- Microsoft 365 Business Standard
- Microsoft 365 Business Premium

[Microsoft 365 pour les employés de première ligne](https://partner.microsoft.com/resources/collection/Microsoft-365-firstline-offer-updates), notamment :
- Microsoft 365 F3 (anciennement Microsoft 365 F1) et Office 365 F3

[Microsoft 365 pour les grandes entreprises](https://www.microsoft.com/microsoft-365/compare-microsoft-365-enterprise-plans), notamment : 
- Office 365 E1
- Microsoft 365 E3 et Office 365 E3
- Microsoft 365 E5 et Office 365 E5

[Microsoft 365 pour l’enseignement](https://www.microsoft.com/education/buy-license/microsoft365), notamment : 
- Microsoft 365 A1 et Office 365 A1
- Microsoft 365 A3 et Office 365 A3
- Microsoft 365 A5 et Office 365 A5

Ce changement est implémenté pour garantir que les nouveaux clients et ceux renouvelant leur abonnement bénéficient de licences pour passer de Kaizala à Teams.

## <a name="csp-incentive-programs-extended-through-fy21-q1"></a><a name="1"></a>Programmes d’incentives CSP prolongés jusqu’au T1 de l’année fiscale 2021

### <a name="categories"></a>Catégories

- Date : 17-06-2020
- Tapez : Fonctions
- État : Bientôt disponible

### <a name="summary"></a>Résumé

Les programmes d’incentives CSP pour l’année fiscale 2020 ont été prolongés jusqu’au 30 septembre 2020. Les programmes de l’année fiscale 2021 commenceront le 1er octobre 2020.   

### <a name="impacted-audience"></a>Public concerné

CSP Direct Bill Partners, Indirect Providers et Indirect Resellers

### <a name="details"></a>Détails

Les programmes d’incentives CSP pour l’année fiscale 2020 initialement prévus pour se terminer le 30 juin 2020 sont prolongés jusqu’au 30 septembre 2020. Les guides des programmes ont été mis à jour pour refléter ce changement. Les tarifs, désormais en place, resteront en vigueur jusqu’au 30 septembre 2020. Les fichiers d’addendum des produits du programme d’incentives CSP ont été mis à jour pour refléter les opportunités de revenus à partir du 1er juillet 2020. Ces fichiers se trouvent dans la [collection de ressources appropriée du programme d’incentives CSP sur MPN](https://aka.ms/partnerincentives).

Les programmes d’incentives CSP pour l’année fiscale 2021 commenceront le 1er octobre 2020. D’autres informations sur les programmes d’incentives CSP pour l’année fiscale 2021 seront fournies le 1er août 2020.


### <a name="next-steps"></a>Étapes suivantes

Partagez les informations contenues dans ce message avec votre organisation afin d’optimiser les opportunités de revenus à partir de juillet 2020. Les partenaires peuvent passer en revue le [dernier guide d’incentives et les ressources de prise en charge](https://aka.ms/partnerincentives).
