## <a name="deprecation-and-retirement-of-existing-get-and-put-qualification-apis-for-the-education-customer-validation-process-by-the-end-of-february-2021"></a><a name="2"></a>Dépréciation et retrait des API de qualification GET et PUT existantes pour le processus de validation des clients Éducation à la fin du mois de février 2021

### <a name="impacted-audience"></a>Public concerné

Partenaires proposant des offres Éducation dans le cadre du programme Fournisseur de solutions Cloud avec l’API Espace partenaires

### <a name="details"></a>Détails 

Il s’agit du suivi des améliorations d’API publiées en décembre 2020. En décembre 2020, de nouvelles API de qualification GET et POST ont été publiées, c’est pourquoi les anciennes API de qualification GET et PUT seront supprimées à la fin du mois de février 2021. À partir de cette échéance, vous devrez utiliser les nouvelles API de l’Espace partenaires GET et POST pour pouvoir acheter des offres Éducation. 

## <a name="next-steps"></a>Étapes suivantes

- Si vous ne l’avez pas déjà fait, passez aux nouvelles API pour une transition réussie et dans les temps.
- Passez en revue les nouveaux changements de l’API Espace partenaires et le [Guide dans les ressources de préparation opérationnelle :  Améliorations du processus de validation des clients Éducation dans l’Espace partenaires](https://partner.microsoft.com/resources/collection/partner-center-edu-validation-enhancements#/).
- Partagez ces informations avec les équipes appropriées de votre organisation et avec vos revendeurs pour les aider à anticiper ces changements.

## <a name="dynamics-365-offers-for-february-2021"></a><a name="1"></a>Offres Dynamics 365 annoncées pour février 2021

### <a name="categories"></a>Catégories

- Date : 04-01-2021
- Offres

### <a name="summary"></a>Résumé

Nouveaux changements dans les offres Dynamics 365 en février 2021.

### <a name="impacted-audience"></a>Public concerné

Partenaires participant aux programmes Fournisseur de solutions Cloud pour les revendeurs indirects, les fournisseurs indirects et les partenaires à facturation directe

### <a name="details"></a>Détails

Microsoft annonce de nouveaux changements dans les offres Dynamics 365 qui prendront effet en février 2021 pour :

- Dynamics 365 Customer Voice Additional Response
- Dynamics 365 Customer Service Insights End of Life
- Dynamics 365 Cloud AX Migration Offer IDs change

**Dynamics 365 Customer Voice Additional Response**

Le 1er février 2021, Microsoft créera une nouvelle offre « Additional Response » avec un droit de réponse de 1 K pour remplacer l’offre existante de 2 K Survey Responses, qui ne sera plus à la vente le 1er février 2021.

Consultez l’onglet « Customer Voice Addl Resp » dans le [document Excel Dynamics CSP Offers-Février 2021](https://partner.microsoft.com/resources/detail/dynamics-csp-offers-february-2021-xls) pour obtenir le détail des références SKU. Pour plus d’informations, consultez la [page d’accueil de Microsoft Dynamics 365 Customer Voice](https://dynamics.microsoft.com/en-us/customer-voice/overview/).

**Dynamics 365 Customer Service Insights End of Life**

Le 1er février 2021, les produits suivants auront le statut Fin de vie :

- Dynamics 365 Customer Service Insights (comprenant « Additional Cases »)
- Dynamics 365 Virtual Agent for Customer Service

L’expérience autonome « Customer Service Insights » est transférée à « Dynamics 365 Customer Service », où les clients peuvent trouver les mêmes fonctionnalités étroitement incorporées dans les expériences du service client.  

Consultez l’onglet « Customer Service Insights » dans le [document Excel Dynamics CSP Offers-Février 2021](https://partner.microsoft.com/resources/detail/dynamics-csp-offers-february-2021-xls) pour obtenir le détail des références SKU. Pour plus d’informations, consultez la [page d’accueil de Microsoft Dynamics Customer Service](https://dynamics.microsoft.com/customer-service/overview/).
