## <a name="self-service-purchasing-for-microsoft-project-and-microsoft-visio-now-available"></a><a name="6"></a>Achat en libre-service de Microsoft Project et Microsoft Visio maintenant disponible

### <a name="categories"></a>Catégories

- Date : 2020-09-15
- Offres/marchés

### <a name="impacted-audience"></a>Public concerné

Partenaires qui vendent des services en ligne aux clients commerciaux

### <a name="details"></a>Détails

L’année dernière, nous [vous avons annoncé](https://partner.microsoft.com/resources/detail/power-apps-power-automate-self-service-pdf) que Microsoft autorise les personnes et les équipes à acheter directement une sélection de services. Aujourd’hui, les utilisateurs du monde entier peuvent acheter des abonnements pour les produits Microsoft Project et Microsoft Visio suivants en libre-service : 

- [Visio Plan 1](https://products.office.com/visio/visio-plan-1?activetab=pivot:overviewtab) et [Visio Plan 2](https://www.microsoft.com/microsoft-365/visio/visio-plan-2?rtc=2&SilentAuth=1&activetab=pivot:overviewtab)
- [Project Plan 1](https://www.microsoft.com/microsoft-365/project/project-plan-1?rtc=1&activetab=pivot:overviewtab) et [Project Plan 3](https://www.microsoft.com/microsoft-365/project/project-plan-3?rtc=1&activetab=pivot:overviewtab)

Remarque :  Actuellement, les fonctionnalités d’achat en libre-service ne sont pas disponibles en Inde, au Brésil et en République démocratique du Congo. Elles ne le sont pas non plus pour les clients éligibles à des offres Secteur public ou Éducation.  Consultez le [FAQ sur les achats en libre-service](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/self-service-purchase-faq?view=o365-worldwide&preserve-view=true) pour obtenir les disponibilités et les mises à jour dans tous les pays.

#### <a name="partner-benefits"></a>Avantages pour les partenaires

Vous pourrez proposer vos services aux personnes et aux équipes sans avoir à passer par un processus d’approvisionnement informatique centralisé pour obtenir les licences requises. Vos services peuvent englober Project et Visio sur la base de scénarios d’intégration avec Microsoft 365 et Power Platform. Avant d’acheter, les clients peuvent aussi essayer avec les versions d’essai de Visio et Project, maintenant disponibles pour [Visio Plan 1 et Visio Plan 2](https://partner.microsoft.com/resources/collection/visio-availability-announcing-trial-offers#/) et [Project Plan 1](https://www.microsoft.com/microsoft-365/project/compare-microsoft-project-management-software).


### <a name="next-steps"></a>Étapes suivantes

- Consultez toutes les ressources de préparation dans la galerie des ressources Operations Readiness : [Capacité d’achat libre-service client](https://partner.microsoft.com/resources/collection/customer-self-serve-purchase#/). -Découvrez plus en détail les opportunités Partenaire avec les solutions Visio et Power Platform dans la session Microsoft Inspire [Rebuild and reimagine business with Microsoft Visio and Power Platform](https://www.microsoft.com/microsoft-365/partners/videos/inspire-visio-power-platform).
- Pour plus d’informations, consultez le [Guide pour gérer les achats en libre-service](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/manage-self-service-purchases-admins?view=o365-worldwide&preserve-view=true).

## <a name="new-skus-bring-the-power-of-project-cortex-and-more-to-customers"></a><a name="5"></a>Les nouvelles références SKU apportent aux clients la puissance de Project Cortex et bien davantage

### <a name="categories"></a>Catégories

- Date : 08/09-2020
- Offres/marchés

### <a name="impacted-audience"></a>Public concerné

Tous les partenaires effectuant des transactions commerciales dans le cadre du programme Fournisseur de solutions cloud

### <a name="summary"></a>Résumé

De nouvelles références SKU pour aider les clients à exploiter la puissance de [Project Cortex](https://resources.techcommunity.microsoft.com/project-cortex-microsoft-365/) et de [Project Turing](https://msturing.org/) seront disponibles pour des transactions commerciales à compter du 1er octobre 2020.

### <a name="details"></a>Détails

Nous sommes heureux d’annoncer la disponibilité au 1er octobre 2020 de nouvelles références SKU, notamment [Project Cortex](https://resources.techcommunity.microsoft.com/project-cortex-microsoft-365/) (Intelligent Content Services) et Recherche Microsoft (avec Microsoft Graph pour Recherche) qui permettront aux utilisateurs de rechercher et de gérer plus efficacement les informations.  

#### <a name="intelligent-content-services"></a>Intelligent Content Services
  
Intelligent Content Services utilise une IA avancée pour automatiser la capture, l’étiquetage et la gestion de contenu. Il s’appuie sur SharePoint pour le traitement de contenu en grands volumes, pour l’extraction des métadonnées pour l’automatisation des processus, et pour la sécurité et la conformité à un niveau détaillé. Intelligent Content Services améliore les expériences de conservation et de découverte des connaissances en envoyant du contenu plus précis dans votre réseau de connaissances. Les nouvelles références SKU d’Intelligent Content Services seront disponibles dans la **liste de prix du 1er octobre** (et dans la préversion de la liste de prix du 1er septembre). 
 
**Remarque :** Intelligent Content Services est un nom temporaire pour ce produit, qui sera probablement renommé après le 1er octobre 2020.
 
**Connecteurs Microsoft Graph pour Recherche Microsoft** Les connecteurs Microsoft Graph permettent aux clients d’indexer des éléments dans des référentiels externes afin de les inclure dans les résultats de Recherche Microsoft. 

À partir du 1 octobre 2020 : 

- Microsoft 365 E5 et Office 365 E5 vont inclure la possibilité d’indexer jusqu’à 500 éléments avec les connecteurs Microsoft Graph pour Recherche Microsoft (non inclus avec A5).  
- Tout utilisateur disposant d’une licence de suite ou d’une licence autonome incluant un plan SharePoint ou OneDrive pourra voir les résultats de la recherche provenant de connecteurs Microsoft Graph pour Recherche Microsoft. 
- Une nouvelle **référence SKU de capacité supplémentaire du connecteur Graph** sera disponible pour l’indexation de jusqu’à 1 million éléments supplémentaires.

## <a name="enhanced-incentives-support-experience-and-ticket-creation-process"></a><a name="4"></a>Amélioration de l’expérience du support Incentives et du processus de création de tickets

### <a name="categories"></a>Catégories

- Date : 02-09-2020
- Fonctions

### <a name="impacted-audience"></a>Public concerné

Tous les partenaires

### <a name="summary"></a>Résumé

L’expérience de support de Partner Incentives a été améliorée avec une meilleure découvrabilité du contenu de l’auto-assistance et un processus de soumission de tickets en ligne plus fluide.

### <a name="details"></a>Détails

Des améliorations ont été apportées à Partner Incentives pour faciliter votre expérience globale du support grâce à une meilleure découvrabilité du contenu de l’auto-assistance et un processus de soumission de tickets plus fluide qui réduit le va-et-vient de la collecte des données pendant le processus de support.

#### <a name="new-capabilities"></a>Nouvelles fonctionnalités

- La nouvelle expérience d’auto-assistance basée sur la recherche est accessible dans l’[Espace partenaires](https://aka.ms/IncentivesSupport) et sur [Microsoft Partner Network](https://partner.microsoft.com/support/?stage=1) pour les comptes autres que ceux de l’Espace partenaires. La nouvelle expérience comprend de nouvelles étapes de dépannage, des Forums aux questions et des liens vers des articles d’auto-assistance appropriés ainsi que dans la page de support de Microsoft Partner Network.
- Lancez une nouvelle demande de support directement dans l’Espace partenaires. Après avoir consulté l’aide disponible sur un sujet spécifique, si vous avez encore besoin d’aide, vous pouvez créer un ticket directement seulement depuis l’Espace partenaires.
- De nouvelles questions relatives aux problèmes sont disponibles pour chaque rubrique d’auto-assistance. Examiner ces questions permet de réduire la majeure partie du va-et-vient qui a lieu au début du ticket de support en fournissant préalablement à l’agent un grand nombre des détails importants.


### <a name="next-steps"></a>Étapes suivantes

- Partagez ces informations avec d’autres personnes de votre organisation. 
- Utilisez la nouvelle expérience de support via l’[Espace partenaires](https://aka.ms/IncentivesSupport) ou sur [Microsoft Partner Network](https://partner.microsoft.com/support/?stage=1) pour tout le support Incentives dont vous avez besoin. 

**Remarque :** Les demandes de support ne sont pas acceptées via des e-mails directs, car cette fonction est désormais hors service.

## <a name="upcoming-new-microsoft-dynamics-365-offers"></a><a name="3"></a>Nouvelles offres Microsoft Dynamics 365 à venir

### <a name="categories"></a>Catégories

- Date : 01-09-2020
- Offres/marchés

### <a name="impacted-audience"></a>Public concerné

Partenaires commercialisant Microsoft Dynamics 365

### <a name="summary"></a>Résumé

Parmi les offres à venir pour les clients Microsoft Dynamics 365, citons la publication de Microsoft Project Operations et d’environnements supplémentaires pour Business Central.

### <a name="details"></a>Détails

Les nouvelles offres suivantes seront disponibles le 1er octobre 2020.

#### <a name="project-operations"></a>Opérations du projet

Le 1er octobre 2020, Microsoft publiera des offres Dynamics 365 Project Operations et supprimera Dynamics 365 Project Service Automation (PSA) du programme Fournisseur de solutions Cloud. Les nouveaux clients devront acheter Dynamics 365 Project Operations. 

Si vous êtes un client existant de Dynamics 365 avec PSA :

- Continuez à utiliser Dynamics365 PSA jusqu’au 1er octobre 2020 ou jusqu’à la fin de votre renouvellement ou contrat le plus récent, selon la première échéance.
- Dynamics 365 PSA sera renouvelé automatiquement pour un an jusqu’au 1er octobre 2021 dans le cadre du programme Fournisseur de solutions Cloud (les contrats pourront donc être prolongés jusqu’au 1er octobre 2022).
- Les licences Dynamics 365 PSA recevront automatiquement Project Operations dans le cadre de leurs licences actives d’ici le 31 décembre 2020.
- Au terme des contrats Dynamics 365 PSA, les clients devront passer à Project Operations qui aura une référence SKU de transition vers PSA de zéro utilisateur minimum.

Pour obtenir des détails sur les ID d’offre, consultez l’onglet Project Operations dans le [document Excel sur les offres Dynamics du programme Fournisseur de solutions Cloud (octobre 2020)](https://partner.microsoft.com/resources/detail/microsoft-dynamics-365-new-ids-offers-csp-october-2020-xls).

- Blog [Introducing Microsoft Dynamics 365 Project Service Automation](https://cloudblogs.microsoft.com/dynamics365/bdm/2020/02/20/introducing-microsoft-dynamics-365-project-operations/)
- Document de formation sur les [nouveautés de Microsoft Dynamics 365 Project Service Automation](https://assetsprod.microsoft.com/mpn/what-is-coming-in-d365-project-service-automation-dyn767pal.pdf)

#### <a name="expansion-of-business-central-offers-into-additional-countries"></a>Expansion des offres Business central dans d’autres pays

Le 1er octobre 2020, Microsoft étendra ses offres cloud Business Central aux pays suivants : Brésil, Irlande, Lituanie, Îles Féroé et Groenland.

Pour obtenir des détails sur les ID d’offre, consultez l’onglet **Environnements supplémentaires** du [document Excel sur les offres Dynamics du programme Fournisseur de solutions Cloud (octobre 2020)](https://partner.microsoft.com/resources/detail/microsoft-dynamics-365-new-ids-offers-csp-october-2020-xls).
