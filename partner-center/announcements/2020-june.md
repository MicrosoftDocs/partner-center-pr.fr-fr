## <a name="now-available-windows-7-year-2-esu-for-csp-partners"></a><a name="6"></a>Windows 7 Year 2 ESU désormais disponible pour les partenaires CSP

### <a name="categories"></a>Catégories

- Date : 29/06/2020
- Tapez : Offres/marchés
- État : Disponible immédiatement

### <a name="summary"></a>Résumé

Les tarifs de Windows 7 Year 2 ESU (Extended Security Updates) sont désormais consultables dans la [Galerie de ressources](https://partner.microsoft.com/resources/collection/windows-esu-csp#/).

### <a name="impacted-audience"></a>Public concerné

Tous les partenaires participant au programme Fournisseur de solutions cloud

### <a name="details"></a>Détails

Le support de Windows 7 a pris fin le 14 janvier 2020. Bien que de nombreux clients soient bien engagés dans le processus de déploiement de Windows 10, Microsoft a conscience que d’autres ne se trouvent pas au même stade dans la mise à niveau. La période de validité de Windows 7 Year 1 ESU prend fin le 12 janvier 2021. Nous savons que certains clients commencent à établir une planification pour l’année prochaine.  

Pour les clients qui prévoient de toujours utiliser des appareils Windows 7 en 2021, nous communiquons dès à présent les tarifs de Windows 7 Year 2 ESU aux partenaires CSP pour aider leurs clients à mûrir leur décision de passer à Windows 10. Pour plus d’informations, consultez la [Galerie de ressources](https://partner.microsoft.com/resources/collection/windows-esu-csp#/).

### <a name="next-steps"></a>Étapes suivantes
Veuillez communiquer ces informations aux contacts concernés de votre organisation. 

Ressources supplémentaires :

- [Page des questions fréquentes (FAQ)](https://support.microsoft.com/help/4527873/itpro-faq-about-end-of-support-for-windows-7)
- [Page de support](https://docs.microsoft.com/deployoffice/windows-7-support)
- [Galerie CSP](https://partner.microsoft.com/resources/collection/windows-esu-csp#/)

## <a name="partner-center-relationship-removal-feature-update"></a><a name="5"></a>Mise à jour de la fonctionnalité de suppression des relations dans l’Espace partenaires

### <a name="categories"></a>Catégories

- Date : 29/06/2020
- Tapez : Fonctions
- État : Bientôt disponible

### <a name="summary"></a>Résumé

Depuis le 29 juin 2020, les fournisseurs et revendeurs indirects de solutions Cloud peuvent supprimer des relations en aval à partir de l’Espace partenaires et n’ont plus besoin pour cela de soumettre un ticket de support de l’Espace partenaires. Cela permet aux partenaires de gagner un temps précieux.

### <a name="impacted-audience"></a>Public concerné

Fournisseurs et revendeurs indirects de solutions Cloud

### <a name="scenarios-supported"></a>Scénarios pris en charge :

- Les fournisseurs indirects qui ont une relation établie avec un revendeur indirect peuvent supprimer cette relation à partir de l’Espace partenaires.
- Les revendeurs indirects qui ont une relation établie avec un client final peuvent supprimer cette relation à partir de l’Espace partenaires.

Dans ces scénarios, ces opérations doivent être effectuées par un agent d’administration de l’Espace partenaires.

### <a name="next-steps"></a>Étapes suivantes :

- Consultez les documents de la galerie de préparation opérationnelle : Mise à jour de la fonctionnalité de suppression des relations de l’Espace partenaires.
- Partagez ces informations dans votre organisation pour permettre à l’équipe compétente de passer en revue et mettre à jour les processus.

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>Changement de la périodicité de facturation par défaut d’annuelle en mensuelle

### <a name="categories"></a>Catégories

- Date : 17-06-2020
- Tapez : Fonctions
- État : Bientôt disponible

### <a name="summary"></a>Résumé

La fréquence de facturation par défaut passe de annuelle à mensuelle dans l’expérience d’achat de l’Espace partenaires.

### <a name="impacted-audience"></a>Public concerné

Fournisseurs de solutions Cloud directs et fournisseurs indirects

### <a name="details"></a>Détails

Le paramètre de fréquence de la facturation détermine la fréquence à laquelle le partenaire est facturé pour les services qu’il a achetés. Les partenaires voient ce paramètre dans l’étape **passer en revue** lors d’un achat dans le catalogue de l’Espace partenaires. Ce paramètre a comme valeur par défaut **annuelle**, car la facturation annuelle a été introduite pour les services basés sur une licence. Suite aux commentaires des partenaires, l’équipe de l’Espace partenaires change cette valeur par défaut à partir du 17 juin de **annuelle** en **mensuelle**, pour qu’elle s’aligne mieux avec ce que les partenaires attendent. Les partenaires peuvent changer cela avant de soumettre une transaction sur la page de révision ou [après une transaction, dans la page Gérer l’abonnement]( https://docs.microsoft.com/partner-center/billing-annual-monthly#change-billing-frequency). Sachez que les offres qui prennent en charge seulement la facturation annuelle continuent d’utiliser cette valeur par défaut. Ce changement de la valeur par défaut est applicable seulement à l’expérience utilisateur du portail Espace partenaires : il n’y a aucune modification du fonctionnement des API de l’Espace partenaires.

Nous espérons que cette modification réduit le risque que le partenaire achète accidentellement des abonnements facturés annuellement.
